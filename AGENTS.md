# AGENTS.md

## Project

Broker Follow-Up Machine GPT documentation, knowledge, and Actions repo.

## Purpose

Maintain the knowledge stack, workflow playbooks, Actions documentation, OpenAPI schemas, and Vercel documentation site for Broker Follow-Up Machine GPT.

## Rules

- Do not commit API keys, client secrets, tokens, or private credentials.
- Do not add real borrower financial data.
- Do not add SSNs, tax records, credit reports, or bank statements.
- Keep borrower-facing language compliance-conscious.
- Avoid approval, eligibility, qualification, rate, term, or guarantee claims.
- Mark human review points clearly.
- Keep markdown files upload-ready for GPT knowledge.
- Keep Action files separated by authentication type.

## Folder conventions

- `/knowledge` = GPT knowledge uploads
- `/actions` = Action planning docs
- `/schemas` = OpenAPI and JSON schemas
- `/examples` = sample outputs
- `/site` = Vercel static pages
- `/prompts` = reusable generation prompts
