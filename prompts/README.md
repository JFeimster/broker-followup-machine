# `prompts/`

This is useful because you’re generating a lot of knowledge files.

Suggested files:

```
prompts/
├── gpt-builder-instructions.md
├── conversation-starters.md
├── testing-prompts.md
└── file-generation-prompts.md
```

Especially useful:

```
file-generation-prompts.md
```

This can store prompts like:

```
Generate a detailed and insightful version of the following suggested file which I can export and add to the GPT’s knowledge: [file_name]
```
