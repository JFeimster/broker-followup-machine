# Brutalist Marketing Site — Step 1: Brand OS Operator

## Purpose

Use this prompt with **Brand OS Operator** to create the brand system, positioning, voice, and visual identity for **Broker Follow-Up Machine GPT**.

This is **Step 1** in the brutalist marketing site prompt chain.

## Prompt

Create the brand OS for **Broker Follow-Up Machine GPT**.

The brand should feel like a neo-brutalist fintech automation tool for loan brokers and business funding brokers.

## Core Promise

Turn a messy broker pipeline into an automated follow-up machine that never forgets a borrower, referral partner, or funding opportunity.

## Required Output

Create:

- Positioning statement
- Homepage headline options
- Subheadline options
- Brand voice
- Color palette
- Typography direction
- CTA language
- Messaging pillars
- Safe language rules
- Phrases to avoid
- Visual identity notes

## Brand Direction

The brand should feel:

- Practical
- Direct
- Revenue-focused
- Operational
- Broker-safe
- Implementation-ready
- Bold enough to stand out
- Trustworthy enough for financial services

## Avoid

Avoid:

- Hype
- Pressure tactics
- Corporate fluff
- Unrealistic promises
- Guaranteed funding claims
- Approval claims
- Eligibility claims
- Lender-specific promises

## Final Instruction

Make the brand feel like a practical revenue-protection system for busy brokers, not a generic AI toy.
