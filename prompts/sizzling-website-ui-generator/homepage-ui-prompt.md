# Sizzling Website & UI Generator Prompt

## Purpose

Use this prompt with **Sizzling Website & UI Generator** to create a bold homepage UI direction, sticky layout, and visual style for **Broker Follow-Up Machine GPT**.

## Prompt

Design a sizzling homepage UI for **Broker Follow-Up Machine GPT**.

## Style

- Neo-brutalist fintech
- Cyberpunk automation dashboard
- Futuristic CRM control center
- Loud, sticky, premium, high-contrast
- Vivid colors that pop
- Matte black, electric cyan, neon lime, hot magenta, safety orange, cyber yellow
- Hard borders, sharp cards, glowing status chips, sticky CTA, dashboard panels

## Required Output

Create:

- Full landing page UI concept
- Hero section
- Sticky navigation
- Sticky CTA rail
- Workflow cards
- Pipeline visualization
- Automation builder section
- Before/after section
- Safety/human review section
- Lead magnet section
- FAQ styling
- Mobile responsive direction

## Aesthetic Target

Make the UI feel like:

**Ramp x Linear x Vercel x neo-brutalist poster design x cyberpunk revenue ops dashboard.**

## Product Context

Broker Follow-Up Machine GPT is a custom GPT for loan brokers, business funding brokers, funding agency owners, independent funding agents, referral partner managers, broker teams, processors, and VAs.

## Core Promise

Turn a messy broker pipeline into an automated follow-up machine that never forgets a borrower, referral partner, or funding opportunity.

## Messaging Direction

Use direct, broker-specific messaging like:

- Stop losing broker deals to forgotten follow-up.
- Your pipeline is not broken. Your follow-up system is.
- Every lead needs an owner.
- Every deal needs a next action.
- Every sequence needs a stop condition.
- Chase missing docs without sounding pushy.
- Revive stalled deals without sounding desperate.
- Build follow-up systems that protect revenue.
- Turn manual follow-up chaos into a machine.

## Design Guardrails

The page should feel like a serious broker follow-up command center, not a generic AI landing page.

Avoid:

- Bland SaaS templates
- Corporate fluff
- Crypto scam energy
- Fake stacks of cash
- Luxury car imagery
- Unreadable neon overload
- Cheesy AI art

## Final Instruction

Make the UI bold enough to stop a scroll, sharp enough to impress a founder, and structured enough to convert a broker.
