# Static Docs Site Page Copy Prompt

## Purpose

Use this prompt with **Notion AI**, **ChatGPT**, **Gemini**, or a custom GPT to generate concise static website copy for the Broker Follow-Up Machine GPT documentation site.

## Prompt

Create concise static website copy for a documentation site for **Broker Follow-Up Machine GPT**.

The site has these pages:

1. `index.html` — main landing page
2. `agents.html` — public-facing GPT/agent explanation
3. `knowledge.html` — knowledge stack browser
4. `actions.html` — GPT Actions categories and authentication types
5. `workflows.html` — workflow playbooks
6. `templates.html` — email/SMS/call/CRM templates
7. `safety.html` — human review and compliance-conscious language
8. `assets.html` — downloadable files, screenshots, examples, and links

For each page, provide:

- Page title
- Meta description
- Hero headline
- Hero subheadline
- 3–5 main sections
- CTA text
- 5 internal links to other pages

## Tone

Practical, direct, operator-focused, clear, and not hype-heavy.

## Context

Broker Follow-Up Machine GPT helps loan brokers, funding agency owners, and broker teams create follow-up systems for:

- New leads
- Missing documents
- Stalled deals
- Declined borrower reactivation
- Funded client renewals
- Referral partner nurture
- Dead lead revival

## Safety

Do not use:

- Approval language
- Qualification language
- Eligibility language
- Rate claims
- Term claims
- Funding guarantee language

Mention human review where appropriate.
