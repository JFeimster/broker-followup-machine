Template purpose

SMS.

Use for short, direct follow-up that needs fast visibility and a simple next action.

When to use it

Best fit

  • Fast follow-up after a missed call
  • Short document reminder
  • Quick check-in after no response

Draft categories

What to draft

  • One-line reminder draft
  • Callback request draft
  • Review-only follow-up draft

Example structure

Simple format

  1. Greeting
  2. Reason for the message
  3. One clear ask
  4. Operator review note

CRM/task usage notes

How to log it

  • Broker should verify any lender-specific wording
  • No automated sensitive messages
