Template purpose

Email.

Use for structured follow-up drafts that need a clear subject, short body, and reviewable next step.

When to use it

Best fit

  • First response after a new lead
  • Document reminders
  • Recap after a call or status change

Draft categories

What to draft

  • Intro email draft only
  • Reminder email draft only
  • Recap email draft only

Example structure

Simple format

  1. Subject line
  2. One-sentence context
  3. Specific ask
  4. Review note

CRM/task usage notes

How to log it

  • Attach the email to the record
  • Set the next follow-up task
  • Note whether operator review is required before send

Human review points

Check before sending

  • Human review required before borrower-facing claims
  • Broker should verify any lender-specific wording
  • Do not auto-send sensitive funding-related messages
