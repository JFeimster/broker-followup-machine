Template purpose

Calls.

Use for live conversation prompts, callback notes, and clear talk tracks that keep the operator in control.

When to use it

Best fit

  • Calling after a quiet lead
  • Following up on missing items
  • Checking in before a review step

Draft categories

What to draft

  • Opening lines
  • Question prompts
  • Callback summary notes

Example structure

Simple format

  1. Reason for the call
  2. Short intro
  3. One question or request
  4. Next-step confirmation

CRM/task usage notes

How to log it

  • Log the call attempt
  • Note the outcome
  • Set the next task or callback reminder

Human review points

Check before sending or speaking

  • Human review required for lender-specific language
  • Broker should verify any sensitive wording
  • Do not use scripts that sound like outcome promises
