Template purpose

Voicemail.

Use for short callback messages that state why you called and what the next action should be.

When to use it

Best fit

  • Call went unanswered
  • Need a concise callback message
  • Need to keep the record moving

Draft categories

What to draft

  • Callback voicemail
  • Reminder voicemail
  • Review-only voicemail

Example structure

Simple format

  1. Name and reason for call
  2. One short context sentence
  3. Callback request
  4. Closing line

CRM/task usage notes

How to log it

  • Record the voicemail attempt
  • Set the next follow-up task
  • Note whether the message needs operator review

Human review points

Check before sending

  • Human review required before borrower-facing messages
  • Broker should verify any lender-specific wording
  • Keep the message draft only until reviewed
