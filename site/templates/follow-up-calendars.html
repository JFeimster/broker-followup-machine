Template purpose

Follow-up calendars.

Use for recurring timing plans that keep touchpoints visible and easy for operators to maintain.

When to use it

Best fit

  • Need a repeatable reminder rhythm
  • Need to map recurring touchpoints
  • Need a reviewable schedule for the team

Draft categories

What to draft

  • Weekly reminder plan
  • Monthly partner check-in plan
  • Post-close reminder plan

Example structure

Simple format

  1. Cadence
  2. Touchpoint type
  3. Owner
  4. Review note

CRM/task usage notes

How to log it

  • Link the calendar to the active workflow
  • Set each touchpoint as a task or reminder
  • Note which items need operator review before send

Human review points

Check before sending

  • Human review required before borrower-facing messages
  • Broker should verify any lender-specific or sensitive wording
  • No automated sensitive messages
