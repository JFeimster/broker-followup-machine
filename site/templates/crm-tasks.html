Template purpose

CRM tasks.

Use for task wording that keeps ownership, timing, and the next action clear inside the record.

When to use it

Best fit

  • Need a follow-up task assigned
  • Need a reminder tied to a record change
  • Need to document review points

Draft categories

What to draft

  • Owner task
  • Reminder task
  • Review task

Example structure

Simple format

  1. Task title
  2. Owner
  3. Due date
  4. Next action note

CRM/task usage notes

How to log it

  • Keep one owner per task
  • Link the task to the active record
  • Use clear language for operator review

Human review points

Check before sending

  • Human review required if the task leads to a borrower-facing send
  • Broker should verify any sensitive note
  • Do not auto-create final copy for sensitive messages
