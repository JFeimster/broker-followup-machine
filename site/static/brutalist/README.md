# Broker Follow-Up Machine GPT - UI Assets

This repository contains the high-conversion, neo-brutalist fintech web app UI for **Broker Follow-Up Machine GPT**.

## Core Concept
A "mission control" dashboard merged with a landing page, designed specifically for loan brokers, business funding brokers, and independent agents to completely automate their pipeline follow-up. 

## Style & Mood
- **Neo-brutalist fintech:** Hard edges, sharp cards, oversized typography.
- **Cyberpunk automation:** Glowing data chips, high-contrast neon accents against matte black.
- **Operational & Sticky:** Designed to feel like a serious tool that solves operational chaos.

## Files Included
- `index.html`: The fully structured front-end landing page.
- `styles.css`: All styling, CSS variables, and layout systems.
- `script.js`: Interactive hover states, sticky nav behaviors, and scroll observer animations.
- `README.md`: Project documentation.

## Running the UI
Simply open `index.html` in any modern web browser. No build steps or package installations are required.

## Color Palette
- Base Background: Matte Black / Charcoal (`#0a0a0c`)
- Accent 1: Electric Cyan (`#00f0ff`)
- Accent 2: Neon Lime (`#39ff14`)
- Accent 3: Hot Magenta (`#ff003c`)
- Warning Alert: Safety Orange (`#ff5e00`)
