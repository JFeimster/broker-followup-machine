# Broker Follow-Up Machine GPT UI

A high-conversion, neo-brutalist fintech / cyberpunk UI for a custom GPT designed for loan brokers and funding agencies.

## Project Structure
- `index.html`: The semantic structure of the high-energy landing page and dashboard.
- `styles.css`: The styling rules defining the neo-brutalist layouts, sharp borders, neon typography, and cyberpunk glow elements.
- `script.js`: Lightweight JavaScript handling dynamic pipeline visualization and scroll interactions.
- `README.md`: This file.

## Design Identity
- **Vibe:** Cyberpunk automation dashboard meets neo-brutalist marketing site.
- **Colors:** Deep Black (`#070707`), Electric Cyan (`#00f3ff`), Neon Lime (`#39ff14`), Hot Magenta (`#ff00ff`), Safety Orange (`#ff5722`).
- **Typography:** Over-sized geometric headings (Space Grotesk), readable functional sans-serif body (Inter).

## How to View
Simply open `index.html` in any modern web browser. No build steps or server required.

## Key Features Implemented
- Sticky top navigation and a persistent bottom CTA rail for maximum conversion behavior.
- High-contrast hero section with poster-style typographic hierarchy.
- Simulated pipeline motion using CSS transitions and a lightweight JS interval.
- Thick, tactile cards representing core GPT workflows and protocols.
- Safety override visual indicators for compliant broker communication.
