Workflow goal

Missing document follow-up.

Goal: keep document requests visible, trackable, and reviewable so the record does not stall while operators wait for uploads.

Trigger

When to start

  • Document request is outstanding
  • Upload stalls after the first reminder
  • Record moves to docs needed

Audience

Who uses it

  • Processors
  • Loan brokers
  • VA and ops support

Step-by-step sequence

Recommended flow

  1. List the missing items.
  2. Draft the reminder email.
  3. Draft the SMS follow-up.
  4. Log the call note if needed.
  5. Set the next reminder if there is no response.

Draft categories

Message types to prepare

  • Email draft only
  • SMS draft only
  • Call script draft only

CRM stages and tags

Suggested record markers

  • Stage: docs requested
  • Stage: docs pending
  • Tag: missing-docs
  • Tag: operator review required

Automation rules

What can be automated

  • Auto-create a reminder task when documents are missing
  • Escalate after repeated no-response windows
  • Do not auto-send sensitive messages without review

Human review points

Where the operator should verify

  • Before referencing sensitive files
  • Before any borrower-facing send
  • Before any lender-specific wording

Metrics to track

Simple measures that matter

  • Documents received
  • Time to complete upload
  • Reminder count before completion
  • Review completion count
