Workflow goal

Dead lead revival.

Goal: bring old records back into a controlled follow-up path without using promotional or promise-heavy language.

Trigger

When to start

  • Record has been inactive for a long stretch
  • Owner wants to revisit an old lead list
  • New context makes a review worth considering

Audience

Who uses it

  • Loan brokers
  • Funding agency owners
  • Ops support managing old records

Step-by-step sequence

Recommended flow

  1. Confirm the record is still useful.
  2. Draft a reactivation email.
  3. Draft an SMS or call follow-up if needed.
  4. Set the next touchpoint after review.
  5. Log the outcome and keep the owner informed.

Draft categories

Message types to prepare

  • Email draft only
  • SMS draft only
  • Call draft only

CRM stages and tags

Suggested record markers

  • Stage: dead lead review
  • Stage: revival queue
  • Tag: old lead
  • Tag: operator verify

Automation rules

What can be automated

  • Queue old records for a draft review
  • Set a reminder if the reactivation path stalls
  • Do not auto-send sensitive messages

Human review points

Where the operator should verify

  • Before any borrower-facing send
  • Before any lender-specific statement
  • Before any message that references old decision context

Metrics to track

Simple measures that matter

  • Revived records
  • Reply count on reactivation
  • Review turnaround time
  • Next-step completion count
