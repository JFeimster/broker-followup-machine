Workflow goal

New lead speed-to-contact.

Goal: respond quickly, capture context, and move the lead into a clean next action without overpromising or skipping review.

Trigger

When to start

  • New lead enters the CRM
  • Lead status changes to first-contact needed
  • No initial response within the target window

Audience

Who uses it

  • Loan brokers
  • Funding agency owners
  • Processors handling first-touch follow-up

Step-by-step sequence

Recommended flow

  1. Confirm the record owner.
  2. Draft the first email or SMS.
  3. Send only after operator review if needed.
  4. Log the response window and next action.
  5. Set the follow-up reminder if there is no reply.

Draft categories

Message types to prepare

  • Email draft only
  • SMS draft only
  • Call opening draft only

CRM stages and tags

Suggested record markers

  • Stage: New lead received
  • Stage: First touch pending
  • Tag: speed-to-contact
  • Tag: first reply needed

Automation rules

What can be automated

  • Trigger a draft task when the lead lands
  • Set a reminder if there is no reply
  • Do not auto-send sensitive funding-related messages

Human review points

Where the operator should verify

  • Before any borrower-facing send
  • Before any lender-specific statement
  • Before any claim that sounds like an outcome promise

Metrics to track

Simple measures that matter

  • Time to first contact
  • First reply count
  • Operator review completion
  • Next-step task completion
