Workflow goal

Referral partner nurture.

Goal: keep partner-channel follow-up steady, useful, and reviewable so inactivity does not turn into lost referrals.

Trigger

When to start

  • Partner has gone quiet
  • Monthly check-in is due
  • Education content needs a refresh

Audience

Who uses it

  • Referral partner managers
  • Partner-channel operators
  • Loan brokers managing repeat sources

Step-by-step sequence

Recommended flow

  1. Check the partner history.
  2. Draft the monthly check-in.
  3. Prepare the education note if needed.
  4. Set the next reminder after review.
  5. Escalate any custom send for operator review.

Draft categories

Message types to prepare

  • Email draft only
  • SMS draft only
  • Call/voicemail draft only

CRM stages and tags

Suggested record markers

  • Stage: partner nurture
  • Stage: inactive partner
  • Tag: monthly check-in
  • Tag: education content

Automation rules

What can be automated

  • Schedule the monthly reminder draft
  • Queue a follow-up if there is no response
  • Do not auto-send sensitive messages

Human review points

Where the operator should verify

  • Before any partner-facing send
  • Before any language that mentions lender-specific details
  • Before any message that sounds like a promise

Metrics to track

Simple measures that matter

  • Partner check-ins sent
  • Partner replies received
  • Inactive partners re-engaged
  • Review completion time
