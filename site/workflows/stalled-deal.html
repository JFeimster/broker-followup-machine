Workflow goal

Stalled deal revival.

Goal: reopen quiet records with a controlled sequence that makes the next action obvious and keeps review points visible.

Trigger

When to start

  • No reply after the expected window
  • Deal sits idle in the CRM
  • Operator flags the record as stalled

Audience

Who uses it

  • Loan brokers
  • Funding agency owners
  • Processors and ops support

Step-by-step sequence

Recommended flow

  1. Check the last recorded touch.
  2. Draft a low-pressure follow-up.
  3. Prepare a call or voicemail draft.
  4. Set a reminder if there is no response.
  5. Escalate to the operator if the response changes the path.

Draft categories

Message types to prepare

  • Email draft only
  • SMS draft only
  • Call/voicemail draft only

CRM stages and tags

Suggested record markers

  • Stage: stalled
  • Stage: re-engagement
  • Tag: no-reply
  • Tag: follow-up review

Automation rules

What can be automated

  • Schedule the follow-up draft after inactivity
  • Trigger a reminder if the next action is overdue
  • Hold for operator review before any sensitive send

Human review points

Where the operator should verify

  • Before any borrower-facing send
  • Before any lender-specific language
  • Before any decline-related or sensitive note

Metrics to track

Simple measures that matter

  • Revived records
  • Response count after re-engagement
  • Days from stall to next action
  • Review completion count
