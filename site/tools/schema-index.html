Tools

JSON Schema draft index.

The utility layer uses JSON Schema draft 2020-12. These schemas are static planning files and should be treated as example-only reference material.

lead-intake.schema.json

Lead intake payload shape with business name, contact details, review flags, and safety flags.

workflow-request.schema.json

Workflow request shape for broker follow-up drafts and channel selection.

template-request.schema.json

Template request shape for email, SMS, call, voicemail, CRM task, objection, and cadence drafts.

safety-review.schema.json

Safety review shape for risk level, scope, and required review state.

crm-stage-map.schema.json

CRM mapping shape for workflow stage and tag suggestions.

webhook-event.schema.json

Webhook event wrapper shape for static event examples.

Standard

Draft version

  • All schema files use draft 2020-12
  • Example fields stay placeholder-only
  • Human review fields stay visible where relevant

Safety note

Review before use

  • Check sensitive document references before sharing
  • Check lender-specific fields before any outward use
