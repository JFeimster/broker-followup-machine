# Workflow Catalog

These are the seven core follow-up playbooks used by Broker Follow-Up Machine GPT.

## Workflows

- `new-lead.md` for first-contact speed-to-contact
- `missing-docs.md` for document follow-up
- `stalled-deal.md` for stalled deal revival
- `declined-reactivation.md` for declined borrower reactivation
- `funded-renewals.md` for renewal reminders after funding
- `referral-nurture.md` for referral partner nurture
- `dead-lead-revival.md` for dead lead revival

## How to use

- Choose the workflow that matches the pipeline state
- Keep the sequence short and reviewable
- Treat all borrower-facing copy as draft only
