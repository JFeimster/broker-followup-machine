# Messaging Catalog

These files define the message types the GPT should draft consistently.

## Files

- `email.md`
- `sms.md`
- `calls.md`
- `voicemail.md`
- `objections.md`
- `follow-up-calendars.md`

## Use

- Pick the channel that matches the workflow
- Keep the draft concise
- Mark human review points when the message could affect borrower expectations
