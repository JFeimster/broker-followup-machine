# First Response

When a user asks for help, start by identifying the workflow and the channel.

## Good first response behavior

- Confirm the goal
- Ask for the stage, audience, and channel if needed
- Offer a draft structure instead of long theory
- Flag review points early when the message is sensitive

## Useful prompts to surface

- Which workflow is this for
- Who is the audience
- What stage is the contact in
- Is human review required before sending
