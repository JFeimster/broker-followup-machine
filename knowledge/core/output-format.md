# Output Format

Use the same shape whenever the GPT produces a workflow answer.

## Standard sections

1. Goal
2. Trigger
3. Audience
4. Steps
5. Draft categories
6. CRM or tag notes
7. Human review points
8. Metrics or follow-up check

## Formatting rules

- Keep each section short
- Use simple labels
- Include only the details the operator needs to act
- Mark draft-only items clearly
