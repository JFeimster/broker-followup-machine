# Examples

This folder holds static example payloads, webhook wrappers, and sample output snippets.

## Purpose

- Show the expected shape of utility data
- Keep examples short, readable, and product-specific
- Support consistency across schemas and GPT output

## Structure

- `payloads/` for request-shaped JSON examples
- `webhooks/` for webhook event wrappers
- `outputs/` for sample workflow output patterns

## Rules

- Use placeholder names only
- Do not include real borrower data
- Keep human review flags visible when relevant
