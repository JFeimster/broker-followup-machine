# Downloadable Assets

## Purpose

Describe static files the GPT can point to without needing an auth path.

## When to use

- You want a catalog of knowledge files, examples, or static references.
- The user needs a planning note for downloads.

## Inputs

- Asset type
- Intended use
- Review boundary

## Outputs

- Asset list
- Use notes
- Access note

## Safety boundary

- Keep assets generic where possible.
- Do not include sensitive borrower data.
