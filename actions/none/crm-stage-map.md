# CRM Stage Map

## Purpose

Sketch CRM stage names and tag ideas without writing to a CRM.

## When to use

- You need a stage map for planning.
- You want a common language for the team.

## Inputs

- Workflow type
- Record status
- Next action

## Outputs

- Stage names
- Tag names
- Owner notes

## Safety boundary

- Keep stage names descriptive, not promotional.
- Do not imply eligibility or outcome certainty.
