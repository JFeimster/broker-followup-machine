# No-Auth Catalog

## Purpose

Document Action ideas that do not need authentication and can stay entirely within planning and drafting.

## Use this when

- The workflow only needs static guidance or generated drafts.
- The operator can manually copy the output into the next step.
- No user account or external system connection is required.

## Inputs

- Workflow name
- Draft request
- Review requirement

## Outputs

- Draft text
- Workflow map
- Safety note

## Safety boundary

- Human review required before borrower-facing claims.
- Do not generate lender-specific statements as final copy.

## Next implementation step

Keep the output limited to drafts, maps, and review notes.
