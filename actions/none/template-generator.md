# Template Generator

## Purpose

Generate a draft template without live transport, auth, or delivery logic.

## When to use

- You need email, SMS, call, or voicemail copy.
- The operator wants a format-first draft.

## Inputs

- Template type
- Workflow context
- Tone

## Outputs

- Draft template
- Example structure
- Review note

## Safety boundary

- Draft only.
- Broker should verify the final wording before send.

## Next implementation step

Pair the output with the appropriate template detail page.
