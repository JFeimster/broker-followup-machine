# Workflow Builder

## Purpose

Outline a workflow sequence without requiring a live system connection.

## When to use

- A broker wants a playbook before implementation.
- The team needs a step-by-step flow for review.

## Inputs

- Workflow name
- Trigger
- Desired sequence

## Outputs

- Ordered steps
- Review points
- Task notes

## Safety boundary

- Keep the draft neutral and operational.
- Human review required before any sensitive send.

## Next implementation step

Convert the workflow into a folder-specific checklist or schema later.
