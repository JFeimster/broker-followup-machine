# Safety Review

## Purpose

Document review-only checks for no-auth planning output.

## When to use

- Before a draft is shared with a borrower-facing operator.
- Before a template is copied into a working sequence.

## Inputs

- Draft text
- Workflow context
- Sensitivity level

## Outputs

- Review note
- Red flag list
- Review to revise, not to send

## Safety boundary

- Human review required before borrower-facing claims.
- Do not convert review notes into final send language automatically.
