# Lead Intake

## Purpose

Draft the first intake response for a new lead without connecting to any external system.

## When to use

- A new lead needs a quick follow-up draft.
- The operator wants a first-touch script before sending.

## Inputs

- Lead source
- Basic contact context
- Response goal

## Outputs

- Draft email or SMS
- Follow-up note
- Review reminder

## Safety boundary

- Human review required before send.
- No borrower-facing promise language.

## Next implementation step

Use this as the first no-auth draft path before any CRM automation is considered.
