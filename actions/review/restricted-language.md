# Restricted Language

## Purpose

List words and phrases that should not appear in borrower-facing copy except as clearly marked terms to avoid.

## Restricted language to avoid

- approval
- qualification
- eligibility
- rate
- term
- guarantee

## Safe framing

Only list these words in a section that explicitly says they are restricted and should be avoided.

## Next implementation step

Use this list when reviewing generated drafts and schema examples.
