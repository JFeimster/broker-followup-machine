# HubSpot

## Purpose

Plan HubSpot-connected Actions for CRM lookups, updates, and planning.

- CRM data needs to be read or updated under user consent.
- The workflow already lives in HubSpot.

- Record type
- Field list
- Review requirement

## Outputs

- CRM planning note
- Field mapping reminder
- Safety checkpoint

- No live sync code here.
