# GoHighLevel

## Purpose

Plan GoHighLevel-connected Actions for CRM and workflow operations.

## When to use

- A record or workflow is managed in GoHighLevel.
- The operator connects the account through OAuth.

## Inputs

- Workflow context
- Record identifiers
- Review requirement

## Outputs

- Planning note
- Record mapping outline
- Safety reminder

## Safety boundary

- Keep the planning doc free of implementation code and secrets.
- Human review required before sensitive messages.
