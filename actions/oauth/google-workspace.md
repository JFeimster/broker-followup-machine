# Google Workspace

## Purpose

Plan Google Workspace-connected Actions for docs, sheets, or drive-based planning.

## When to use

- The workflow needs a user-connected Google document or sheet.
- A draft must be saved or reviewed in a workspace.

## Inputs

- Target file type
- User connection
- Review note

## Outputs

- Workspace planning note
- File interaction outline
- Human review checkpoint

## Safety boundary

- Keep it planning-only.
- No secrets or auth code.
