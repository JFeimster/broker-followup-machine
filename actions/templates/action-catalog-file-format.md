# Action Catalog File Format

Use this format for each Action catalog file.

# [Action File Name]

## Purpose

What this group of Actions is for.

## Authentication Type

None / API Key / OAuth

## When to Use

- Use case 1
- Use case 2
- Use case 3

## When Not to Use

- Risk 1
- Risk 2
- Risk 3

## Action Catalog

### Action Name: [Action Name]

**Purpose:**
What this Action does.

**Authentication:**
None / API Key Basic / API Key Bearer / API Key Custom / OAuth

**Best Used For:**
When the GPT should choose this Action.

**User Inputs Needed:**

- Input 1
- Input 2
- Input 3

**API Inputs Needed:**

- Field 1
- Field 2
- Field 3

**Expected Output:**
What the Action returns.

**Human Review Required:**
Yes/No, and why.

**Safe Usage Notes:**
Boundaries, privacy rules, and stop conditions.

**Example User Request:**
“Create a follow-up task for this borrower.”

**Example GPT Behavior:**
What the GPT should do before/after using the Action.
